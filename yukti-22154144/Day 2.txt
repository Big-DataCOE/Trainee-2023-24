//Forking and Branches 

Fork : It is used to copy Author repository link to make changes in it .
//Steps to fork :
	1. open Github and Open repository which need to fork .
	2. fork and create that trainee repo 
	3. On Trainee repo copy it's link and clone it on your git .
	4. Make changes to your Trainee folder and push it on github .
